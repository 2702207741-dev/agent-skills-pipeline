# Compatibility Table: v1.1.0

| Surface | Status | Install Path | Trigger Verification |
|---|---|---|---|
| Hermes Agent | Supported | `~/.hermes/skills/<category>/<name>/SKILL.md` | `skill_view(name)` resolves and prompt trigger loads |
| Claude Code | Supported | `~/.claude/skills/<name>/SKILL.md` | New session discovers skill and prompt trigger loads |
| Codex | Supported | `~/.codex/skills/<name>/SKILL.md` | Skill list includes item and description trigger loads |
| Cursor Skills | Supported | `~/.cursor/skills/<name>/SKILL.md` | Description-matching prompt loads the skill |
| Cursor Rules | Limited | `~/.cursor/rules/<name>.md` | Use only for tiny always-loaded guidance |

## Version Compatibility

| Component | Minimum | Notes |
|---|---:|---|
| Python | 3.10 | Required for validation, release, benchmark, marketplace, and graph scripts |
| Bash | 4.x | Required only for `scripts/install.sh`; Python marketplace works without Bash |
| Registry schema | 1 | `skills.json` must include project `version`, release policy, and skill versions |
| Release artifact | v1.1.0 | Generated by `scripts/create_release.py` |
