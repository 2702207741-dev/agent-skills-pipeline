## Trusted Distribution

The release directory contains a zip artifact plus sidecars:

- `our-skills-v1.5.0.manifest.json`
- `our-skills-v1.5.0.sha256`
- `our-skills-v1.5.0.sbom.json`
- `our-skills-v1.5.0.provenance.json`
- `our-skills-v1.5.0.sig`

`scripts/verify_release.py` verifies sidecar hashes, SBOM shape, provenance subject hashes, canonical provenance signature, install/update/rollback behavior, and audit logging.
