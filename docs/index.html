our-skills v3.0 ecosystem entry

Portable agent skills with quality, safety, and replay evidence.

List, install, update, rollback, diagnose, review, and evaluate skills without learning the repository layout first.

Marketplace

The local marketplace index exposes every skill with version, platform paths, dependencies, status, owner, and quality score.

python scripts/marketplace.py list
python scripts/marketplace.py install --platform codex --target-root ~/.tmp --apply
python scripts/marketplace.py update --platform codex --target-root ~/.tmp --skill skill-review-workflow --apply
python scripts/marketplace.py rollback --platform codex --target-root ~/.tmp --skill skill-review-workflow --apply

Doctor

The doctor checks registry and index sync, source metadata, dependencies, platform paths, trigger metadata, and installed versions.

python scripts/marketplace.py doctor --platform codex --target-root ~/.tmp --strict

Quality

Maintainers can scan pass rates, regressions, owners, last updates, lifecycle status, graph islands, cycles, and stage coverage.

Replay
