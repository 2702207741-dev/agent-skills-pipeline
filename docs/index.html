our-skills v4.0.0 external-adoption release

Portable agent maintenance infrastructure another repository can adopt.

Install and replay one skill in five minutes, gate repository-owned skills with one Action, and verify a complete issue-to-release workflow.

Marketplace

The local marketplace index exposes every skill with version, platform paths, dependencies, status, owner, and quality score.

./our-skills list
./our-skills install --platform codex --target-root ~/.tmp --apply
./our-skills update --platform codex --target-root ~/.tmp --skill skill-review-workflow --apply
./our-skills rollback --platform codex --target-root ~/.tmp --skill skill-review-workflow --apply

Doctor

The unified doctor checks repository health, external adoption contracts, registry and index sync, security, dependencies, and installed state.

./our-skills doctor
./our-skills doctor --platform codex --target-root ~/.tmp --strict

Five-Minute Quickstart

Use an isolated target to preview installation, apply one skill, diagnose it, and replay a hash-bound execution record.

./our-skills quickstart --platform codex --target-root "$PWD/.quickstart-home" --apply
