# Code of Conduct

This project welcomes contributors who help keep the skill ecosystem useful,
safe, and evidence-driven.

## Expected Behavior

- Be respectful and constructive.
- Assume good intent while still reviewing changes carefully.
- Keep feedback focused on behavior, evidence, security, and maintainability.
- Disclose conflicts of interest when reviewing a contribution.
- Protect private data, credentials, and sensitive operational details.

## Unacceptable Behavior

- Harassment, insults, threats, or discriminatory language.
- Publishing private information without permission.
- Submitting known malicious behavior, hidden network dispatch, or secret leaks.
- Repeatedly bypassing the review, safety, or replay evidence requirements.

## Enforcement

Maintainers may edit, close, or remove issues, pull requests, comments, or
contributions that violate this code of conduct. Serious or repeated violations
may result in being blocked from participating in the project.

## Reporting

Report conduct concerns privately to the repository maintainer. For security
issues, follow `SECURITY.md`.
